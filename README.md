# Créa Maillage Audit

Créa Maillage Audit est une extension WordPress permettant d’analyser le maillage interne de votre site et d’identifier rapidement les contenus orphelins (articles sans liens entrants).

L’analyse est manuelle, mise en cache, et n’impacte jamais les performances du front-end.

---

## 🇫🇷 Français

### 🎯 Objectif

Cette extension permet :

- D’analyser le maillage interne global
- D’identifier les articles orphelins
- De visualiser les liens internes entrants et sortants
- De filtrer par Articles, Pages ou Articles + Pages
- D’éviter tout impact sur les performances du site

Elle est particulièrement utile pour :

- Optimisation SEO
- Audit de contenu
- Amélioration de la structure interne
- Détection des pages isolées

---

### ⚙️ Fonctionnement

- L’analyse est déclenchée manuellement
- Les résultats sont mis en cache
- Aucun traitement n’est effectué côté front-end
- Scan complet de tous les contenus publiés

---

### 🔒 Sécurité

- Vérification des permissions utilisateur
- Protection via nonce WordPress
- Aucune exécution publique

---

### 🚀 Installation

1. Téléversez le dossier du plugin dans `/wp-content/plugins/`
2. Activez l’extension dans l’admin WordPress
3. Accédez à l’outil depuis le menu d’administration
4. Lancez l’analyse

---

### 🧠 Cas d’usage SEO

Identifier les articles orphelins permet :

- D’améliorer le PageRank interne
- De renforcer les contenus stratégiques
- De corriger les erreurs de maillage
- D’optimiser l’architecture du site

## 🇬🇧 English

### 🎯 Purpose

This plugin allows you to:

- Analyze your website’s internal linking structure
- Detect orphan posts (posts without incoming internal links)
- Visualize internal incoming and outgoing links
- Filter by Posts, Pages or Posts + Pages
- Avoid any front-end performance impact

---

### ⚙️ How it works

- Manual scan only
- Results are cached
- No front-end execution
- Scans all published content

---

### 🔒 Security

- User capability checks
- WordPress nonce protection
- No public execution

---

### 🚀 Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate it from the WordPress admin panel
3. Open the tool from the admin menu
4. Run the scan

---

### 🧠 SEO Benefits

Finding orphan posts helps you:

- Improve internal PageRank
- Strengthen strategic content
- Fix internal linking gaps
- Optimize site architecture
